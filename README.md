# Chocolate-Sales-SQL-Analysis
Project Overview

This project analyzes a chocolate sales dataset using SQL Server to uncover meaningful business insights and evaluate overall sales performance. The analysis focuses on key performance indicators (KPIs) such as total revenue, order volume, product performance, country-wise sales, sales channel performance, and revenue trends over time.

Throughout the project, I used a variety of SQL techniques including data aggregation, filtering, sorting, grouping, subqueries, date functions, and aggregate functions to answer common business questions. The goal is to demonstrate practical SQL skills that are commonly used by data analysts to transform raw sales data into actionable insights.

This project is part of my data analytics portfolio and showcases my ability to write clean SQL queries, perform exploratory data analysis (EDA), and generate reports that support data-driven decision-making.
